How it works

Three views, one workflow.

📝 Notes

Full markdown editor with syntax highlighting and live preview. Use [[wikilinks]] to connect notes together, creating a personal knowledge graph.

Markdown Wikilinks Live Preview

🔗 Graph

Visual network of all your notes and tasks. Drag nodes to rearrange, zoom in and out, and draw connections by tapping source then target.

Drag & Drop Zoom Visual Links

📊 Board

Kanban-style task board. Add tasks, track progress, move between columns. Every task is also a node in your graph.

Kanban Drag Columns Progress

🔍 Search

Full-text search across all notes and tasks. Results update as you type, grouped by type for quick navigation.

Full-text Real-time Offline
